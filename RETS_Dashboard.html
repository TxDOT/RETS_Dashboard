Release notes:

  • Updated Year fields to add 2020

  • Added “Activity” field

    New field contains values for “TxDOTConnect, Minute Order, CRI, Sprints,” etc.

  • Added “Activity” filter in TOC pane

    Filter records by Activity

  • Added character counters to text boxes

    All large text boxes have character counter to prevent entering more characters than will save

  • Added validation to attribute pane fields

    Users cannot enter incorrect combinations and cannot leave certain fields blank

  • Added "Advanced Query" tab to TOC pane

    Allows user to enter custom SQL queries that are not available using only filters

  • Added "Application Options" tab to TOC pane

    This contains Log Off button and new Version Info button

  • Standardized alerts and modal behavior

  • API Updates

    Updated Esri JS API to 3.32
